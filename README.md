## 📌 Project Overview

This project presents a **comprehensive financial analysis dashboard** built using **Power BI** to analyze daily household transactions.  

The dashboard provides deep insights into:
- Income vs Expenses
- Category-wise spending patterns
- Savings trends
- Monthly financial performance
- Expense distribution analysis

The objective of this project is to transform raw transactional data into meaningful, actionable financial insights using data modeling, DAX, and interactive visualizations.

 [Daily Household Teansactions\_Data.csv](https://github.com/Kshitija-Agrawal/Finance-Dashboard-PowerBI/blob/main/Daily%20Household%20Transactions.csv) — Sample Dataset

- Category (Food, Rent, Travel, etc.)
- Income / Expense Indicator
- Amount
- Payment Mode
- Description (optional)

## ⚙️ Data Preparation Steps

### 1️⃣ Data Cleaning (Power Query)
- Converted `Order_Date` / `Transaction_Date` from Text → Date format
- Removed null values
- Standardized category names
- Ensured correct data types
- Created Month & Year columns

### 2️⃣ Data Modeling
- Created relationships
- Generated calculated columns
